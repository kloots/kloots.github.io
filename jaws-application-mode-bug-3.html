JAWS ARIA application mode example #3

Overview

In this example JAWS will consistently switch between forms mode and virtual mode.

  1. Press F6 to move focus to the button inside the application context.
  2. Notice JAWS switches into forms mode.
  3. Press Tab to move focus to the button inside the document context.
  4. Notice JAWS switches into virtual mode.
  5. Press Shift + Tab to move focus back to the button inside the application context.
  6. Notice JAWS switches back into forms mode.
